### Dog Tech

### 背景（製品開発のきっかけ、課題等）
　AIの発達に伴い，Siriのような音声データによる自然言語処理のアプリ開発が盛んになっています.しかし，会話をするのは人間だけではありません.そこで，犬と飼い主の意思疎通が出来るようなアプリを考えました.

### 製品説明


### 特長

#### 1. 犬の感情がわかる

#### 2. 犬種の選択可能
