## 製品概要
### Dog x Tech

### 背景（製品開発のきっかけ、課題等）
　AIの発達に伴い，Siriのような音声データによる自然言語処理のアプリ開発が盛んになっています.しかし，会話をするのは人間だけではありません.そこで，犬と飼い主の意思疎通が出来るアプリを考えました.

### 製品説明(流れ)
#### 1. 犬種選択(大型，小型)
#### 2. 鳴き声録音

### 特長
